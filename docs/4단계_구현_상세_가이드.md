# 04. 구현 상세 가이드 (Implementation Guide)

본 문서는 IvansTerm의 주요 기능을 구현하기 위한 상세 로직과 가이드를 설명합니다.

---

## 1. SSH 터미널 WebSocket 구현
- **Backend**: FastAPI의 `WebSocket` 엔드포인트에서 SSH 세션을 열고, PTY(Pseudo Terminal)를 할당하여 양방향 스트리밍.
- **Frontend**: xterm.js 인스턴스를 생성하고 WebSocket에 연결. 키 입력은 WS로 전송, 수신된 데이터는 터미널에 렌더링.
- **Resize**: xterm.js의 `onResize` 이벤트 발생 시 WS를 통해 Backend에 cols/rows 전달, PTY 크기 동기화.
- **Reconnect**: 네트워크 순단 시 자동 재연결 로직 (지수 백오프 적용).

---

## 2. SFTP 파일 편집 구현
- **파일 트리**: 접속 시 루트 디렉토리를 조회하고, 트리 노드 클릭 시 하위 디렉토리를 Lazy Loading으로 가져옴.
- **파일 열기**: 파일 트리에서 파일 클릭 → REST API로 내용 조회 → Monaco Editor에 로드. 파일 확장자에 따라 언어 모드 자동 감지.
- **파일 저장**: `Ctrl+S` 단축키 → REST API로 현재 에디터 내용을 리모트 서버에 SFTP Write. 저장 상태 피드백(Saving → Saved) UI 표시.

---

## 3. 동적 패널 레이아웃 구현
- **상태 관리**: PanelContext에서 좌/우 영역별 패널 목록(type, id, props)을 관리.
- **패널 추가**: `+` 버튼 클릭 시 에디터/터미널 타입 선택 모달 → 해당 영역에 패널 추가 (최대 3개/영역, 6개/전체 제한).
- **패널 이동**: 드래그 앤 드롭으로 영역 간/내 재배치. 이동 시 상태 배열 업데이트.
- **리사이징**: react-resizable-panels 등 라이브러리를 활용하여 패널 간 경계선 드래그 리사이즈.
- **레이아웃 저장**: 패널 구성 변경 시 자동으로 `layout_state` (JSON)를 DB에 저장하여 재시작 시 복구.

---

## 4. 파일 전송 구현
- **Drag & Drop 업로드**: HTML5 Drag & Drop API로 파일 수신 → FormData로 REST API 전송 → Backend에서 SFTP Put 실행.
- **다운로드**: 파일 트리 우클릭 컨텍스트 메뉴 → REST API 호출 → Backend에서 SFTP Get → StreamingResponse로 클라이언트에 전달.
- **진행률**: 대용량 파일 전송 시 청크 단위 전송 + 하단 상태바에 프로그레스 바 표시.

---

## 5. 접속 정보 암호화 전략
- **라이브러리**: `cryptography` 패키지의 `Fernet` 대칭 암호화 사용.
- **Master Key**: 환경 변수(`ENCRYPTION_KEY`)로 관리. 최초 실행 시 자동 생성하여 `.env`에 저장.
- **암호화 대상**: SSH 비밀번호, Private Key 경로 또는 내용.
