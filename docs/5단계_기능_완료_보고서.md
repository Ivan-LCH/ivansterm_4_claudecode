### 1.1 SSH 터미널
- [ ] FastAPI WebSocket + SSH PTY 연결 구현
- [ ] xterm.js 기반 프론트엔드 터미널 UI
- [ ] 터미널 리사이즈(cols/rows) 동기화
- [ ] 네트워크 재연결 로직

### 1.2 리모트 파일 에디터
- [ ] SFTP 기반 파일 트리 브라우저 (Lazy Loading)
- [ ] Monaco Editor를 이용한 파일 열기/편집
- [ ] Ctrl+S 즉시 저장 (SFTP Write)
- [ ] 파일 확장자별 언어 모드 자동 감지

### 1.3 동적 패널 레이아웃
- [ ] 좌/우 영역 1:1 분할 기본 레이아웃
- [ ] 패널 동적 추가/제거 (한쪽 최대 3개, 전체 최대 6개)
- [ ] 패널 드래그 이동 (영역 간/내 재배치)
- [ ] 경계선 드래그 리사이징
- [ ] 레이아웃 상태 저장 및 복구

### 1.4 파일 전송
- [ ] Drag & Drop 업로드 (로컬 → 리모트)
- [ ] 우클릭 다운로드 (리모트 → 로컬)
- [ ] 전송 진행률 상태바 표시
- [ ] 전송 이력 기록

### 1.5 접속 관리
- [ ] 접속 정보 CRUD (등록/수정/삭제)
- [ ] 접속 정보 암호화 저장 (Fernet)
- [ ] 접속 테스트 기능
- [ ] 멀티 세션 탭 전환 (다중 접속 세션 관리)
- [ ] 앱 재시작 시 마지막 작업 환경 자동 복구
